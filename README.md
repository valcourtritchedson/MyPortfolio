## How to Customize

- **Colors**: Edit CSS variables in `:root` selector in style.css
- **Content**: Update text, projects, and skills in index.html
- **Animations**: Adjust animation timing in CSS keyframes or JavaScript
- **Fonts**: Change the font-family in the `body` CSS rule
